# Reflection: AI-Assisted Development

## 🧠 What I Learned
I learned how to effectively manage "Context Drift." As the project grew, I had to provide specific file snippets to the AI to ensure it understood the Hexagonal boundaries I had established.

## ⚡ Efficiency Gains vs Manual Coding
AI-assisted coding reduced my UI development time by roughly 70%. Writing Tailwind 4.0 configurations and complex Recharts components manually would have taken hours; with Gemini, I could iterate on the "Banking View" in minutes.

## 🛠️ Improvements for Next Time
Next time, I would initialize the project with a stricter `tsconfig` from the start. Most of my final-hour "friction" came from TypeScript version mismatches (v5.9) and Prisma v7 breaking changes that the AI initially treated with older patterns.
